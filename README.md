# Social-Media-Scraper

Run "pip install -r requirements.txt"
1. First from Instagram, Run app.py as "streamlit run app.py"
2. Next From Linkedin, Run app_streamlit.py as "streamlit run app_streamlit.py"
3. Last From X/backend, run app.py as "python app.py"
       then From X/frontend/myapp, run react as "npm start"

   Then open index.html
